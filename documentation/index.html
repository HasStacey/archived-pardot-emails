Tune Email Documentation

We have created a suite of templates to help bring our email marketing inline with our branding and message. We have created these templates to be as stylistically unified as possible but fit the vast needs of our marketing team. Below are graphic standards the creative team has complied to help keep everything looking unified and polished.

Please do not copy an existing template to create a new, unique template. If you need a one-off template and none of these fit your needs, please reach out to Dimitri with your request: dimitri@tune.com

Finding the templates

The templates are located in Pardot under Marketing > Emails > Templates. In the filter text box, type "2015" to bring up the list of email tempates.

General

Best Practices

The templates we provided have more modules than what we proposed in our initial designs. You may need to duplicate or remove some modules to fit your needs.

Note: HO and TUNE template layouts and configurations are the same except for the logo in the header.

Buttons

Buttons should be configured as 1 or 2 buttons only. Please avoid using 3 or more buttons in one module.

Events Module

Please use the event logo as the event image. Events should come in a pair or in a group of three; never in a group of 5.

Character Limits: Headlines

Headlines and titles should be set at a maximum of two lines.

There are two exceptions to the 2 line limit. The first is for the Single Column Micro-Post, which has a limit of 1 line. The second is for the Two-Column Micro-Post, which has a limit of 3 lines.

Character Limits: Paragraphs

Paragraphs should be set at a maximum of four lines and use ellises where necessary.

Copy/Paste formatting issues

All fonts should be unaltered -- please do not change the color, case or the size of the text. It's okay to use bold and italic text. Your workflow may consist of copying a block of text and pasting it in the email. Image Library

Swapping the header and icons

Some of the templates feature swappable header and icon images. We have built a library of images approved for email use. If you need a custom image, please reach out to dimitri@tune.com who will manage your request.

Changing the header image or icon

  • From the new email, click on the header image. An Image Editor dialog will open.
  • From the drop down list, scroll until you see "2015-email-bg" and select the image you would like
  • The process is the same for the icons. Approved icons start with "2015-email-icon"

Adding a new header image or icon

If you are going to upload new files for use, it's important to adhere to the naming convention of our files for these templates.

Header Images: File name should start with "2015-email-bg-"

Icons Icons should be transparent PNG files. Icon file names should start with "2015-email-icon-"

  • Navigate to Marketing > Content > Files
  • Upload the new file.
  • The file will now be available for use in the drop down list in the templates
